# OpenClaw Version Notes & Migration Guardrails

- Verify against installed dist: `$(npm root -g)/openclaw/dist/config-*.js`
- Re-check key enums after upgrades (`nativeSkills`, visibility, binding peer kinds).
- Run: `openclaw doctor`, restart gateway, probe channels.
